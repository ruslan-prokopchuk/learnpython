Основные моменты о языке.

Python - интерпритируемый язык.

Плюсы языка: 
    1. Это интерпритируемый язык.
    2. Высокоуровневый язык.
    3. Кроссплатформенный.
    4. Простой язык.
    5. Встраиваемый скриптовый язык.
    6. Лёгкое написание мелких скриптов-программ.
    7. Огромная библиотека классов.
Минусы языка:
    1. Низкая скорость выполнения работы скрипта. (C или же C++ работает быстрее, чем данный язык)
    2. Динамическая типизация языка. (Долго работает с большими скриптами)
С помощью языка можно создать:
    1. Системные утилиты.
    2. Графические приложения. (GUI API)
    3. Веб-приложения. (Фреймворки)
    4. Веб-сценарии. (Сетевые операции)
    5. Интеграция компонентов.
    6. Работа с базой данных.
